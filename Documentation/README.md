# KISRAK-M1 Documentation

Welcome to the official documentation for the KISRAK-M1 project. This document aims to provide a comprehensive and in-depth understanding of the KISRAK-M1 motor driver, covering its design philosophy, detailed hardware architecture, underlying physics, control mechanisms, fabrication guidelines, and validation procedures.

KISRAK-M1 is designed as a high-performance, open-source motor driver, specifically tailored for micro-robotic applications like Micromouse. Our goal is not only to present "what" was built but also to elaborate on "why" specific design choices were made and the fundamental physical principles that govern its operation.
